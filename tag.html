Sobre TAG's

As tag's são comandos utilizados para que o computador entenda o que aquele trecho de código representa em nosso HTML. Elas podem ser:

HEADER: utilizada para descrever o topo do seu site. Geralmente lá ficam armazenadas as informações de menu e logo.

SECTION: é utilizada para dividir seu site em assuntos específicos.

DIV: são divisórias para seu código. Você pode colocar "etiquetas" (classes ou id) nas suas divisórias para facilitar na hora de colocar um estilo.

STYLE: a tag style é utilizada para você inserir um estilo (em formato CSS) dentro do seu arquivo HTML.
